Durant ce projet, j'ai appris à utiliser l'HTML et le CSS. Voilà, ce que j'ai retenu !

HTML

L'html est le langage principalement et quasiment toujours utilisé pour coder des sites. Il est souvent couplé avec le CSS et le JavaScript ( je n'ai pas utilisé ce dernier pour la création de ce site).
J'ai retenu les points suivant sur ce langage de programmation

  • Ce langage de progammation utilise des balises, importantes pour définir n'importe quoi.
  • Pour incérer des photos ou d'autres éléments côte à côte, on peut créer un tableau avec "table"
  • Ne pas oublier de lier le CSS au début du code !
  • Pour faciliter la mise en page avec le CSS, on peut mettre en place des divisions ( avec div ) pour pouvoir gérer les éléments en groupes
  • On définit toujours un code en deux partie :
    1. head : renferme les précisions sur le code qui ne seront pas visible par l'utilisateur.
    2. body : partie principale du code. Contient donc toute les parties que l'utilisateur pourra voir.

CSS

Le CSS est un outil pratique pour rendre le site esthétique. Pour cela, il faut créer une nouvelle page de code et ne pas oublier de la lier au fichier HTML qu'elle complète. J'ai retenu les éléments suivants sur ce langage :

  • On définit les éléments à l'aide des divisions préalablement faites avec des # (ils ne sont cependant pas necessaire.)
  • "Table" et ses attributs comme "tr" et "td" ne fonctionnent pas avec un # devant.
  • Pour positionner des éléments centrés, on peut utiliser "margin", "float" et "left" ou "right". Ainsi, on vire complétement l'élément de la page avant de le recentrer. Cela marche peut importe la taille de la fenêtre (j'ai cependant réussi cette manipulation avec 4 entités).
  • L'outil "hover" permet une modification de l'élément lorsque qu'il est survolé par la souris !
